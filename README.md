# 🚀 Java Full Stack Mini Projects Portfolio

This repository contains a collection of **4 mini projects** developed as part of my learning journey as a **Java Full Stack Intern**.
Each project demonstrates practical implementation of **frontend technologies (HTML, CSS, JavaScript)** and core web concepts.

---

## 📂 Projects Included

### 🔹 1. 📍 Location Tracker

* Retrieves user's **current location (Latitude & Longitude)**
* Uses **Geolocation API**
* Provides a link to view location on **Google Maps**

**💡 Features:**

* Real-time location detection
* Simple UI with user interaction
* Browser permission handling

---

### 🔹 2. 📖 Dictionary App

* Fetches meaning of a word using a **free Dictionary API**

**💡 Features:**

* Search any English word
* Displays meaning and example
* Handles invalid input gracefully

---

### 🔹 3. 🍽️ 3D Rotating Carousel

* A **3D image carousel** similar to a mall display stand

**💡 Features:**

* Circular image arrangement
* Smooth **3D rotation using CSS**
* Professional UI with animation

---

### 🔹 4. 💼 Portfolio Website

* A **multi-page personal portfolio website**

**💡 Pages:**

* Home
* About
* Contact

**💡 Features:**

* Responsive layout
* Card-based design
* Animations (hover & fade effects)
* Form validation using JavaScript
